To run plasp translated instances located in asp-translations folder:

	./clingo ../asp-translations/TPP-plasp-translated/meta-encoding.lp ../asp-translations/TPP-plasp-translated/instance_23.lp -c horizon=11

	./clingo ../asp-translations/TPP-manually-translated/domain.lp ../asp-translations/TPP-manually-translated/p23.lp -c steps=11
